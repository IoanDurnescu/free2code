HTML Strong Importance <strong> Element

By Christopher Kielty, Published Jan 23, 2019

Use <strong> to assign strong importance to a selection of text. The result is typically displayed as bold text, but not necessarily. This is similar to <b>, except that <strong> has semantic meaning of being of strong importance, while <b> just means let's make this text bold. And actually, because of the awesomeness that is CSS, neither <strong> nor <bold actually need to be bold at all.
